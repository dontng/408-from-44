# 项目进度（2026-06-15）

真题驱动的 408 选择题刷题机。设计与背景见 Claude 长期记忆 `project-408-study-system` / `project-408-situation`。

## 已完成
- **studio 刷题台**：`bash studio.sh` 一键上线本地网页（127.0.0.1:8408，免代理）。**和室·侧栏+题流**版式：左细栏（186 倒计时 / 今日两门 / 进度圆点 / 点击跳题），右侧当天**全部题一条纵向流**，按 科目·章节 分组成块，题面图裁白铺满、ABCD 小方块横排、答完原地判分。遗忘曲线状态存 `review/state.json`（可多机同步）。无官方答案的题自动转**自判模式**，不会误判。
- **历史回顾·左右键翻日（已落地）**：刷题台顶部一左一右翻日箭头，**键盘 ←/→** 直接翻。今天=可答的实时页(最右)，往左翻到过去任一有做题记录的日子，只读展示当天做过的题(图+答对/答错)，侧栏小结与圆点同步成当天。后端 `/api/day?d=YYYY-MM-DD`(`build_day`)+ `progress_days`/`day_nav`；判分时把每题当天对错写进 `_progress[日].res`，故回顾不受日后复习改判影响(改动前的旧记录无 res，回退用 last_ok)。战况诊断视图里不响应方向键。
- **战况诊断面板（已落地）**：左栏「▸ 战况诊断」进入(或 `#stats` 直达)。四项反馈：①冲刺预测分(各科正确率×408满分求和，/150，目标120) ②今日小结(正确率+今日错题章节) ③弱项诊断(科目·章节正确率升序，最弱在前) ④错题本(最近做错的题按章节列出+题面图)。后端 `build_stats` / `/api/stats`。
- **调度器（已落地）**：遗忘曲线 + **科目轮转**（`数据结构+网络`/`组成+操作系统` 按天交替，新题两门均衡）+ **分块→交错→随机** 三阶段（`SCRAMBLE_PHASES`：8/31 前分块、11/30 前交错、之后全真模拟）。复习题(due)按曲线始终全出；新题受轮转/分块约束。无 tags 的题按题号位置兜底归科。参数全在 `studio.py` 顶部。
- **题库：17 年全部完整，680 题（2009–2025 各 40 题）** → `bank/<年>/qNN.png`，已全部**裁白**（`tools/imgtrim.py`，切题/补切时自动调用）。
- **题图字号一致化（已落地）**：源 PDF 字号/页宽差异大、裁白后每张图宽窄不一，旧版网页按宽度铺满会让屏幕字号差到一倍。`tools/imgnorm.py` 用**连通域量出的汉字像素高**(按年取中位数，即用户真正看到的字大小)算出每题**显示宽** → `review/imgnorm.json`；网页按此宽显示(不铺满)，屏幕汉字高恒为 11px(全年实测 10.1–11.0)。为容下归一化后的图(中位 621px)，版式主栏放宽到 1180/gap64。`studio.py` 仅读 json(保持纯标准库)，**dispW 在进程启动时载入——改图/重算后必须重启服务**(`pkill -f studio.py; bash studio.sh`)，光刷新网页不生效。**新切/重裁某年后须重跑** `python3 tools/imgnorm.py [年份…]`。
- **三套切题工具**（`tools/`）：
  - `slice_paper.py` — 文字层按题号坐标切题（适用文字层正常的年份）。
  - `slice_paper_ocr.py` — OCR 定位题号坐标切题（适用坏编码/纯扫描/图片页年份）。OCR 只用来找题号，题面成品仍是原始渲染图，正文 OCR 错误不影响结果。平局取靠后起点，避开"考生须知 1~5"污染。
  - `patch_q.py` — 手工补切漏题：`scan` 列某页左边距行的 y 比例，`cut` 按页+上下比例切单题，`cut2` 跨页题竖向拼接（依赖 Pillow）。
- **答案**：`answers/2009–2024.txt` 已就位（16 年）。

## 常用命令
```bash
bash studio.sh                                       # 启动刷题台
python3 tools/slice_paper.py     真题pdf/XXXX.pdf XXXX   # 文字层切某年
python3 tools/slice_paper_ocr.py 真题pdf/XXXX.pdf XXXX   # OCR 切某年(扫描/坏编码)
python3 tools/patch_q.py XXXX scan <页>                 # 找漏题的 y 坐标
python3 tools/patch_q.py XXXX cut  <题号> <页> <y0> <y1>          # 补切单题
python3 tools/patch_q.py XXXX cut2 <题号> <页> <y0> 1.0 <页+1> 0 <y1>  # 跨页题
python3 tools/imgnorm.py                              # 重算题图显示宽(切/裁图后跑)
```
