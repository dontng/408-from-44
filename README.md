# 408-from-44

> 从混沌到清晰。

一个人的考研 408 备考工具——真题驱动，把"做题"接上"记忆"和"诊断"，
专治"看着都会、一考就废"。名字里的 44 是上一战的分数，也是这次的起点。

## 目标

408（计算机学科专业基础综合）选择题，靠真题 + 间隔复习 + 弱项诊断练到考场能稳定输出，而不是反复刷输入。

## 功能

- **真题题库**：2009–2025 共 17 年、680 道选择题，按题切成单题图（`bank/`）。
- **本地刷题台**：`bash studio.sh` 一键起网页，和纸界面、倒计时、答完即判（无官方答案的题转自判）。
- **调度器**：遗忘曲线间隔复习 + 科目轮转（两门/天）+ 分块→交错→随机三阶段，越临考越像全真模拟。
- **战况诊断**：待解清单（标"没懂"的题，只堆不清）、弱项诊断（科目·章节正确率）、错题本、冲刺预测分。
- **工具链**：切题 / OCR 切题 / 跨页补切 / 题面裁白 / 答案抽取（`tools/`）。
- **学习数学模型**：每日题量由遗忘曲线和时间预算推导，不拍脑袋（`docs/学习数学模型.md`）。

## 用法

```bash
bash studio.sh          # 启动刷题台 (127.0.0.1:8408)
```

进度见 `STATUS.md`。真题 PDF 与答案 PDF 本地留存、不入库。
