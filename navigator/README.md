# 生活时间账本

这里不是「提高效率」的项目，而是你的生活节律记录。它追踪工作、睡眠、恢复、自主时间与长期目标之间的真实关系，帮助你做有依据的选择，而不是把每小时都折算成考试产出。

## 目录结构

| 目录 | 用途 |
| --- | --- |
| `sleep/` | 睡眠与恢复数据：Apple Watch 导出的夜间汇总与分析 |
| `work/` | 工作考勤：2026 年 1—7 月班次、加班、夜班与公出 |
| `dontng/` | 个人底账：身份、四战目标、现实约束、AI 对你的历史理解 |
| `assistant_context/` | 时间助理运行上下文：已知事实、增量输入协议、决策记录 |
| `yangruikee/` | 委托管理研究：从一位「账号由助理管理」的创作者反推可迁移的时间管理模型 |

## 阅读顺序

1. [关于 dontng 与目标](dontng/README.md) — 先理解「这个人是谁、在争取什么」

4. [最近 30 天睡眠](sleep/最近30天睡眠分析-2026-07-22.md) — 当前负荷的近距离观察

时间助理的持续上下文在 [assistant_context/](assistant_context/README.md)，日常变化按 [增量输入协议](assistant_context/增量输入协议.md) 补给。

## 原则

1. 原始健康导出不提交仓库；只保留理解节律所必需的派生数据。
2. 数据用来理解约束和选择权，不用来贴「自律/懒惰」的标签。
3. 恢复、通勤、吃饭、社交与学习同为真实生活；没有记录不意味着浪费。
4. 每条规则必须可回链到数据或复盘；不因一次状态好坏永久固化。
