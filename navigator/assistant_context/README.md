# 时间助理上下文库

这个目录保存时间助理做出判断所需的持续上下文。不是日记，不是完整人生档案。

## 用途

1. 把已确认的事实、目标、边界和模型假设从聊天上下文抽离，降低遗忘与幻觉风险。
2. 新消息以增量方式更新状态：不要求固定频率填表，任意一句现实变化、截图或结果都可以进入系统。
3. 每次决策区分**事实 / 推断 / 未知**；推断可被新数据推翻，未知不脑补。

## 文件

- [时间助理当前状态](时间助理当前状态.md)：已确认事实、已知模式、待验证点。
- [增量输入协议](增量输入协议.md)：如何向助理补给新信息，以及助理的默认响应格式。
- [决策与校准记录](决策与校准记录.md)：重要决策、实际结果与规则修订。

## 工作原则

- 管理的是现实容量，不是名义空闲时间。
- 工作、睡眠、通勤、吃饭、洗漱、恢复、社交和学习都是真实占用。
- 计划偏离不自动解释为意志力问题；优先检查睡眠债、班次、任务入口、环境摩擦与估时误差。
- 助理负责协调、追踪、重排、优先级和复盘；人保留目标变更、主观感受和最终执行权。
- 原始健康数据不提交公开仓库；这里只保留决策所需的派生结论。
