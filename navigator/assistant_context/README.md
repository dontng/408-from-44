# 时间助理上下文库

这里保存的是对“动态时间助理”持续判断有用的上下文，不是日记，也不是完整人生档案。

## 用途

1. 把已确认事实、目标、边界和模型假设从聊天上下文抽离，降低遗忘与幻觉风险。
2. 让新消息以增量方式更新状态：你不必固定频率填表，任意一句现实变化、截图或结果都可以进入系统。
3. 每次决策都区分**事实 / 推断 / 未知**；推断可被新数据推翻，未知不能被脑补成事实。

## 文件

- [current-state.md](current-state.md)：当前稳定事实、已知模式、待验证点。
- [inbox-protocol.md](inbox-protocol.md)：如何把新的“i+1”轻量补给进来，以及助理的默认响应。
- [decision-log.md](decision-log.md)：重要决策、结果和规则修订。只写会改变以后判断的条目。

## 工作原则

- 管理的是现实容量，不是名义空闲时间。
- 工作、睡眠、通勤、吃饭、洗漱、恢复、社交和学习都是真实占用。
- 不把计划偏离自动解释为意志力问题；优先检查睡眠债、班次、任务入口、环境摩擦与估时误差。
- 助理默认负责协调、追踪、重排、优先级和复盘；人保留目标重大变更、主观身体感受和最终执行权。
- 原始健康导出与精确敏感记录不提交到公开仓库；这里只保留为决策所需的派生结论。
