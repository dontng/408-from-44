**06.** 下列关于计算机系统层次结构的说法中，正确的是( ) 。

**07.** 关于编译程序和解释程序，下列说法中错误的是 ( ) 。

**08.** 只有当程序执行时才将源程序翻译成机器语言，并且一次只能翻译一行语句，边翻译边执行的是 ( ) 程序，把汇编语言源程序转换为机器语言程序的过程是( ) 。 I. 编译  II. 目标  III. 汇编  IV. 解释

- B. IV、II

- D. IV、III
