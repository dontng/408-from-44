## 共性OCR错误规律（供参考）
- 国定 → 固定（分区）
- 庶拟 → 虚拟
- 了/刀/严/妃 → T/B/F/B（变量名）
- 工 → I 或 II（罗马数字，需看语境）
- 饥俄/饥钱 → 饥饿
- 信和号 → 信号
- 扒栈 → 栈
- JI/O → I/O
- 生子/合子/谷子/禾子/徐子 → 筷子（哲学家问题）
- 题号 01 反复出现 → 通常是 15 的OCR错误
- 题号 04 突然出现在40+区间 → 通常是 45 的OCR错误
- 选项末尾 [ → 删除
- I_I → I、II / I_II → I、III（下划线=顿号，后续字母往往也错一位）
