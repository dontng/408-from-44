# Time management

这里不是“提高效率”的项目，而是你的生活节律账本。它记录工作、睡眠、恢复、自主时间与长期目标之间的真实交换，帮助你作出有依据的选择，而不是把每一小时都折算成考试产出。

    sleep/       睡眠与恢复：原始导出 → 可复核的夜间汇总 → 生活节律的底线
    work/        工作、加班、夜班与被占用时间（已导入 2026-01—07 截图）
    study/       各科实际投入、启动成本、交付量（待导入）
    decisions/   根据数据作出的生活与安排决策（待建立）

原则：

1. 原始 Apple Health 导出含高度私密的健康信息，不提交到仓库；只提交为理解节律所必需的派生汇总。
2. 数据首先用来理解约束、代价和选择权，不用来给人贴“自律/懒惰”的标签。
3. 恢复、通勤、吃饭、家务、社交与自主学习同属真实生活；没有记录前，不把晚间空档自动判为浪费。
4. 每条时间规则必须能回链到数据或后续复盘，不能凭一次状态好坏永久固化。
