# 以太网 CSMA/CD：竞争、冲突与二进制指数退避

```text
A ───── τ ─────► B ───── τ ─────► A
             2τ：最远两站的往返传播时间
```

## CSMA/CD：共享以太网

共享、半双工介质：多站共用一条链路；同一时刻只能有一个站成功发送。

- **CS — Carrier Sense（载波监听）**：发送前由物理层检测介质上是否存在载波；忙→持续监听；空闲→立即发送（1-坚持）。
- **MA — Multiple Access（多点接入）**：A、B 相距较远时，彼此信号尚未到达本地，两端会同时判定空闲并发送。
- **CD — Collision Detection（冲突检测）**：发送期间继续监听；检测到冲突→停发 + Jam（强化碰撞信号）→二进制指数退避→再侦听、重发。

## 二进制指数退避：冲突后

- 冲突次数 `k`：`1 ≤ k ≤ 10` → `r` 从 `0 ～ 2^k−1` 中随机取。
- `10 < k < 16` → `r` 从 `0 ～ 1023` 中随机取（指数封顶）。
- `k = 16` → 本帧丢弃；不再重传。
- 等待时间 = `r × 争用期`；题目问“最大”时，`r` 取区间右端。

`k` 以题干定义为准：题干若按“已发生 k 次冲突”计就直接代入；教材若按重传次数计，不能自行多加或少减一次。

## 争用期 `2τ`：时间、速率、距离

- 以太网：争用期 = `2τ = 512 bit times`。
- `b = bit`；`B = Byte = 8 b`；`1 Mb = 10^6 b`。
- `1 s = 10^3 ms = 10^6 μs = 10^9 ns`；`ms` 毫秒；`μs/us` 微秒；`ns` 纳秒。

### 由速率换算争用期

- `10 Mb/s = 10×10^6 b/s = 10 b/μs` → `1 bit time = 0.1 μs = 100 ns`。
- `100 Mb/s = 100 b/μs` → `1 bit time = 0.01 μs = 10 ns`。
- `10 Mb/s`：`512 bit times = 512×0.1 μs = 51.2 μs = 0.0512 ms`。

### 由距离判断是否能检测冲突

- 若传播速度为 `2×10^8 m/s = 200 m/μs`：`2500 m → τ=12.5 μs，2τ=25 μs`。
- 能冲突：两端在 `τ` 内未收到对方信号时同时发送；能检测：帧发送时间必须 `≥ 2τ`。
- `10 Mb/s` 最小帧 `512 b`：发送需 `51.2 μs > 25 μs`，故 2500 m 场景来得及检测。

### 最大等待时间

11 次冲突、10 Mb/s、求最大等待：`r=1023`；`1023×51.2 μs = 52.3776 ms`。
