# 2024-36

隐藏站 B 收到的是 AP 发出的 CTS。此时 RTS、等待 CTS 的 SIFS 和 CTS 本身都已经结束，NAV 只需覆盖 CTS 之后的 `SIFS + DATA + SIFS + ACK`；DIFS 用于最初争用信道，也不计入。

数据帧发送时间为：

```text
1998 B × 8 / 54 Mb/s = 296 μs
```

因此 CTS 中的 NAV 为：

```text
28 + 296 + 28 + 2 = 354 μs
```

选 B。A 少算了一个 SIFS；C 在正确结果上又加入一个 SIFS 和已经结束的 RTS，即 `354+28+3=385 μs`；D 再把交换开始前的 DIFS 加到 C，得到 `385+128=513 μs`。计算边界见 [802.11 CSMA/CA：RTS/CTS 与 NAV](../../knowledge/network/02-data-link/wifi-csma-ca-rts-cts-nav.md)。
