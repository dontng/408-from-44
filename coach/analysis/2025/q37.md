# 2025-37

内网主机向 Internet 发送 UDP 报文时，NAT 至少把 IP 首部的源地址改成公网地址；在多主机共享公网地址的常见 NAT 中，还要改 UDP 的**源端口号**，用转换后的端口标识这条内网会话。因此 I 会被修改。

UDP 的**目的端口号**是外部服务器的服务端口，不能因 NAT 而改，II 不成立。NAT 只是改写地址和端口，并未给 UDP 报文段增添字节，故 III“总长度”不变。A 把“经过 NAT”误当成“报文段变长”。

UDP 校验和不仅覆盖 UDP 首部和数据，还计算含源 IP 地址的伪首部。源端口和源 IP 任一改动都会使原校验和失效，IV 必须修改。

所以被修改的是 I、IV，选 **B**。见 [NAT：地址转换、端口转换与 UDP 校验和](../../knowledge/network/nat.md)。
