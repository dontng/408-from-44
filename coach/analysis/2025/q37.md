# 2025-37

NAT 路由器用转换后的公网端口标识这条内网通信，因此 UDP 的源端口号会被修改，I 正确。目的端仍是原来的 Internet 应用，目的端口号不能改变，II 错误。

UDP 总长度等于 8 B 首部加数据。NAT 只是替换源端口等定长字段中的数值，没有增加 UDP 报文段的字节数，所以总长度不变，III 错误。路由器转发时重新封装的是链路层帧，不计入 UDP 总长度。

UDP 校验和还要计算含源 IP 地址的伪首部。NAT 改了源 IP 地址和 UDP 源端口号，原校验和随之失效，IV 正确。

所以只有 I、IV，选 **B**。A 错在把路由器重新封装链路层帧理解成 UDP 报文段变长；C、D 则把不会改变的目的端口号列了进去。字段变化见 [NAT：地址与端口转换、UDP 校验和](../../knowledge/network/03-network/nat.md)。
